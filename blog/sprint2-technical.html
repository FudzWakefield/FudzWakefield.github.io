What are the best practices associated with using Classes vs ID's?

ID - An ID selector is a name preceded by a hashtag character (#). This is used when specifying one element in the styling process. The image below shows how the ID selector is used within HTML and CSS. The h1 tag displays the name "Fudz". In CSS, it is displayed #Fudz. The green box on the right is the end result.

Class - A Class selector is a name preceded by a fullstop (.) This is used when specifying multiple elements in the styling process. The image below shows how the Class selector is used within HTML and CSS. The Class name is "Jokes" and is shown above the h2 tags. In CSS, it is displayed as .Jokes. The green boxes on the right are the end result.

The difference between ID (#) and Class (.) is that ID is used when you want to style one element and Class (.) is used to style multiple elements.
